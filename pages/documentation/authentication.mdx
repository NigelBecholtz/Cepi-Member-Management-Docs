## API Key

All API calls require a valid API key. The key can be provided in three ways:

| Method | Example |
|--------|---------|
| Query parameter | `?api_key=YOUR_API_KEY` |
| Header `Authorization` | `Authorization: Bearer YOUR_API_KEY` |
| Header `X-API-Key` | `X-API-Key: YOUR_API_KEY` |

## Obtaining an API Key

API keys are created in the Admin Dashboard under **API Keys**. Store the key securely; it is only shown once after creation.
