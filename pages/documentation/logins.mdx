## Overview

The CEPI Member Management application has two separate login areas, each with different access levels.

**Emails:**
- support@optimasys.es
---

## Organisation Login

**URL:** `https://mmcepi.optimasit.com/login`

**Who uses it:** Organisation users (members of a CEPI or non-CEPI organisation).

**Access after login:**
- Import members (Excel/CSV)
- Export members (CSV/Excel)
- View organisation data

**Credentials:** Organisation accounts are created by an admin. Contact your administrator if you need access.

---

## Admin Login

**URL:** `https://mmcepi.optimasit.com/admin/login`

**Who uses it:** Administrators only.

**Access after login:**
- Dashboard
- Manage organisations
- Create and manage API keys
- Create admin accounts
- Manage organisation accounts

**Credentials:** Admin accounts are created by another admin. Contact an existing administrator if you need admin access.

---
