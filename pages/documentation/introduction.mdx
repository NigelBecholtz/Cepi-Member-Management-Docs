## Welcome

Welcome to the CEPI Member Management API documentation. This API enables server-to-server integration for checking whether an email address is registered as a member of a CEPI organisation.

**Powered by Zudoku**

## Base URL

```
https://mmcepi.optimasit.com/api
```

## Overview

The CEPI Member Management API provides:

- **Check Member** — Verify if an email is registered as an active member
- **Organisation details** — CEPI vs non-CEPI, MM-CEPI status, member statistics
