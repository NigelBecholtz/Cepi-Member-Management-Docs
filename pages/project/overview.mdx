## Introduction

The CEPI Member Management project is a Laravel application for managing CEPI members and organisations. This document helps developers quickly understand how the project works.

## Architecture

```
┌──────────────────────────────┐
│      cepi-laravel            │
│   (Laravel 12, PHP 8.2)      │
│   • API check-member         │
│   • Admin dashboard          │
│   • Import/Export            │
└──────────────┬───────────────┘
               │
    ┌──────────▼──────────┐
    │   MySQL Database    │
    │   (cepi)            │
    └─────────────────────┘
```

## Project Structure

| Path | Technology | Purpose |
|------|------------|---------|
| `cepi-laravel/` | Laravel 12, PHP 8.2 | API, check-member endpoint, admin dashboard, import/export |
